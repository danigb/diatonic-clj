# diatonic

> Music notation library for Clojure

Work in progress. Not ready to be used yet.

```clojure
(require '[diatioic.note :as note])

(note/pitch-class "fx6") ; => "F##"
(note/midi "C4") ; => 60
(note/transpose "C4" "5P") ; => "G4"
```
